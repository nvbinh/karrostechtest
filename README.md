# KarrosTech Test

### 1. Introduction
The project is built in the ReactJS framework. There are the automation tasks in Webpack version 4.

The project included the following third parties:
+ `React slick`: The carousel at the top banner
+ `Axios`: Connect to the moviedb API
+ `Redux`: States management
+ `Redux Logger`: A logging tool
+ `dotenv webpack`: Exposes environment variables (in the `.env` file) that are referenced in your code
+ `eslint`: A tool for identifying patterns in ECMA/Javascript code
+ `postcss`: CSS styles
+ `webpack`: Build the web app

### 2. Build the web application
`yarn install`

### 3. Start the development environment
`yarn dev`
